# algorithm
一些算法学习的实验代码

## datastructure
数据结构：Trie树、优先队列、二叉查找树

### Mobile
BitMap处理手机号去重，支持海量手机号数据去重，处理时间毫秒级，占用内存极少，使用外部存储保存字典数据

### Trie
字典树，主要实现了下面功能：      
1、将字符串插入字典树     
2、便利字典树将所有出现的字符串打印出来    
3、统计字典树词频TOPK   
4、统计字符串出现次数

## resume
多线程分析导出的51job或者智联招聘简历邮件，自动筛选重要字段导出txt文本。   
经测试本机运行分析400个51job简历，并写入txt文本大约耗时4s

## number
实现了一个大数类型，支持加法和减法

## sort
排序算法
排序测试类支持高斯随机数 -- 2018.9.1  
新增堆排序  --2018.9.23

## filesplit
大文件切割，并去重
